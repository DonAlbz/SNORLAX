Manuale per l'installazione dell'applicativo Snorlax.
Per procedere con l'installazione installare l'applicazione XAMPP,
tale applicazione verra' utilizzata per hostare il proprio sito dinamico.
Successivamente prendere copia del progetto da GitHub tramite funzione Pull origin
ed inserirla nella cartella xampp/htdocs.
Aprire il programma XAMPP e avviare il database.
Aprire il proprio terminale e digitare al suo interno i seguenti comandi per la creazione del database:
mysql -u root
Incollare il codice nel terminale del file database.sql

Snorlax PHP + MySQL

Avvio rapido
1. Copia la cartella in htdocs
2. Avvia Apache e MySQL
3. Importa database.sql in phpMyAdmin
4. Controlla config.php
5. Apri index.php
